# Heart Disease Prediction Web Application

A multi-model machine learning web application built with Python and Streamlit to predict heart disease risk metrics based on clinical patient parameters.

---

## 🚀 Project Overview
This project focuses on predicting the presence of heart disease using patient data. It trains three separate machine learning models to compare accuracy and deployment performance:
* **Logistic Regression** * **Decision Tree** * **Support Vector Machine (SVM)**

The trained models are deployed via an interactive **Streamlit web dashboard** that allows users to input medical metrics individually or upload a batch file to get instant risk predictions.

---

## 🛠️ Tech Stack & Features
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Streamlit, Pickle
* **Interactive UI:** Dynamic inputs for clinical metrics (Age, Sex, Cholesterol, Max Heart Rate, etc.).
* **Batch Processing:** A bulk upload feature supporting `.csv` test files for rapid data evaluation.
